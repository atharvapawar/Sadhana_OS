- Stages 1–2: Doctrine + Inner Science foundation.
- Stages 3–6: Roadmap + Path & Practice foundation.
- Stages 7–11: Path & Practice + Inner Science, with safety gates.

- Stages 17–18: Doctrine + Path & Practice integration.
