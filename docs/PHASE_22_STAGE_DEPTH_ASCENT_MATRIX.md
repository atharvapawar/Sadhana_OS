This phase deepens the existing 18-stage river without adding new main pages or changing the sacred structure.

## Purpose

The existing 18 stages remain the central spine:

`Misidentification → Right Seeing → Purification & Foundation → Body-Prana Stabilization → Interiorization & Meditation → Living Realization`

Phase 22 adds a deeper, connected layer so each stage can be understood through:

- Sanskrit meaning
- one-sentence essence
- river metaphor
- why this stage comes here
- what goes wrong if skipped
- doctrine connection
- inner science connection
- practice connection
- common mistake
- bridge to next stage
- kosha relation
- bhuta / element relation
- chakra-language relation
- vayu relation
- body-practice direction
- purification target
- healing meaning
- safety gate
- glossary terms

## Integration principle

The ascent matrix does **not** claim one rigid universal chart where one chakra, one element, or one kosha mechanically equals one stage. It is a traditional yogic interpretive map that helps users see how many rivers meet:

- kosha map
- element map
- vayu map
- chakra language
- asana/practice direction
- purification/healing target
- safety guidance

## Safety principle

Advanced topics are visible but protected. The matrix must never become a DIY instruction manual for:

- forced kumbhaka
- strong bandha-mudra combinations
- kundalini activation attempts
- chakra forcing
- strong shatkarma
- extreme fasting
- advanced kriya
- risky inversions

## Implementation guardrail

Do not add a separate main page for this matrix. It belongs inside the existing stage detail experience so the 18-stage river remains the spine.
