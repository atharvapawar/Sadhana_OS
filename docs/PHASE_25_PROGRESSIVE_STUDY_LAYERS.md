# Phase 25 — Progressive Study Layers

Phase 25 adds a guided reading layer inside each existing stage detail page. It does not add new main pages, routes, or content hierarchy changes.

## Purpose

The stage pages now contain deep knowledge: intelligence cards, stage-depth architecture, ascent matrix, cross-linking, practices, safety notes, and glossary bridges. Phase 25 prevents information overload by giving each seeker a clear reading sequence.

## The five layers

1. **Quick orientation** — read essence and stage signal first.
2. **Sequence logic** — understand why the stage comes here and what goes wrong if skipped.
3. **Body-prana-inner science map** — connect kosha, bhuta, vayu, chakra language, and practice direction without rigid literalism.
4. **Safe practice gate** — choose the safe application before advanced intensity.
5. **Integration and cross-linking** — connect the stage to Doctrine, Inner Science, Path & Practice, Glossary, and daily life.

## Accessibility model

The layer interface uses real buttons with `aria-expanded` and `aria-controls`. Each opened panel is exposed as a labelled `region`. This follows the WAI-ARIA accordion pattern while keeping native focus order simple and mobile-friendly.

## Safety rule

The feature does not add any advanced kriya, kundalini, kumbhaka, bandha, mudra, shatkarma, or inversion instructions. Advanced curiosity is routed back to the protected practice zone.
